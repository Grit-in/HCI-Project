# SFHub - Street Fighter 6 Learning Platform

A comprehensive learning platform for Street Fighter 6 players, featuring character guides, frame data, matchup strategies, and community features.

## 🎮 Project Overview

This project is implemented in **two versions**:

### 1. **Original Version (HTML/CSS/JavaScript)**
- **Complete implementation** with all required features
- **Direct file opening** - just double-click `index.html`
- **All professor requirements** fulfilled
- **Professional design** and functionality

### 2. **Vue.js Version (Exact Replica)**
- **Identical design** to the original version
- **Same functionality** with Vue.js reactivity
- **Component-based architecture**
- **Modern development** practices

## 🚀 How to Run

### Original Version (Recommended for Submission)
```bash
# Option 1: Direct opening (Easiest)
# Simply double-click on index.html in your file explorer

# Option 2: Local server
npx serve . --port 3000
# Then open http://localhost:3000
```

### Vue.js Version (Exact Replica)
```bash
# Install dependencies
npm install

# Run development server
npm run serve
# Then open http://localhost:8080
```
